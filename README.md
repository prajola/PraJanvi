# Prajanvi - Interactive Portrait Slider ✨

A premium, interactive website designed as a gift for Priya. It features a high-fidelity horizontal slider, swipe transitions, and a responsive portrait handle.

## Features
- **Premium Design**: Curated color palette, smooth gradients, and modern typography.
- **Interactive Slider**: Navigate via mouse drag, touch swipe, keyboard arrows, or navigation dots.
- **Micro-animations**: Subtle entrance animations for every page and interactive elements.
- **Responsive**: Adapts gracefully to both desktop and mobile screens.

## 🚀 How to Run Locally

### Option 1: Using the Run Script (Recommended)
1. Open your terminal.
2. Navigate to this directory.
3. Run the following command:
   ```bash
   bash run.sh
   ```
4. Your default browser will open to `http://localhost:8000`.

### Option 2: Manual Start
If you prefer to start the server manually:
```bash
python3 -m http.server 8000
```
Then visit [http://localhost:8000](http://localhost:8000) in your browser.

## 📁 Project Structure
- `index.html`: Main structure and content.
- `styles.css`: Premium styling and animations.
- `script.js`: Interactive slider and sparkle logic.
- `assets/`: High-resolution images and backgrounds.

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with custom properties and animations.
